# 🟣 Chamba — Branding & Design System

## 🧠 Concepto de Marca

**Chamba** es una plataforma que conecta personas con oportunidades reales de trabajo y servicios, eliminando la incertidumbre del proceso.

La marca busca **dignificar el trabajo**, tanto profesional como de oficio, ofreciendo un entorno donde cualquier persona —desde un programador hasta un albañil— pueda mostrarse de forma **profesional, confiable y transparente**.

---

## 🎯 Valores de Marca

* **Confianza** → Perfiles verificados, reseñas reales, contacto seguro.
* **Transparencia** → Respuesta garantizada en postulaciones laborales.
* **Profesionalismo** → Todos los perfiles se ven de alta calidad.
* **Empatía** → Lenguaje cercano, inclusivo y accesible.
* **Accesibilidad** → Diseño claro para todas las edades y niveles digitales.

---

## 🗣️ Tono de Comunicación

* Claro y directo (sin tecnicismos innecesarios)
* Cercano pero profesional
* Con guiños culturales argentinos (sin exagerar)
* Evitar lenguaje corporativo frío

### Ejemplos:

✔ “Encontrá trabajo con respuesta real”
✔ “Profesionales verificados cerca tuyo”

❌ “La mejor plataforma del mercado”
❌ “Soluciones innovadoras disruptivas”

---

## 🎨 Paleta de Colores

```css
--color-primary: #A01EED;
--color-black: #000000;
--color-gray: #94A3B8;
--color-white: #FFFFFF;
```

### Uso:

* **Primario (#A01EED)** → Botones, acciones, destacados
* **Negro (#000000)** → Títulos, textos importantes
* **Gris (#94A3B8)** → Textos secundarios
* **Blanco (#FFFFFF)** → Fondo principal

⚠️ Regla: no usar más de 3 colores simultáneamente en pantalla.

---

## 🔤 Tipografía

### Principal:

* **Poppins**
* Uso: títulos, encabezados

### Secundaria:

* **Inter**
* Uso: párrafos, inputs, UI

### Jerarquía:

* H1 → 32px / Bold
* H2 → 24px / SemiBold
* H3 → 18px / Medium
* Body → 14–16px / Regular

---

## 🧩 Estilo Visual

* Minimalista
* Limpio y aireado
* Bordes redondeados (8px–12px)
* Uso moderado de sombras
* Iconografía simple y lineal
* Evitar saturación visual

---

## 🧱 Componentes UI

### 🔘 Botones

```css
.btn-primary {
  background: #A01EED;
  color: #FFFFFF;
  border-radius: 8px;
}

.btn-primary:hover {
  background: #8B18D1;
}

.btn-primary:active {
  background: #6F12A8;
}
```

---

### 🧾 Inputs

```css
.input {
  border: 1px solid #E5E7EB;
  border-radius: 8px;
}

.input:focus {
  border-color: #A01EED;
}
```

---

### 🏷️ Tags / Chips

* Fondo gris claro
* Texto oscuro
* Bordes redondeados completos (pill)

---

## 📇 Tarjetas (Cards)

* Fondo blanco
* Bordes suaves
* Sombra leve
* Información jerárquica clara

Contenido típico:

* Nombre
* Rubro
* Etiquetas
* Rating
* Distancia
* Acciones

---

## 🗺️ Mapa

* Base: OpenStreetMap
* Marcadores:

  * Color principal (#A01EED)
  * Variante PRO con glow o badge
* Tooltip:

  * Nombre
  * Rubro
  * Distancia
  * CTA “Ver perfil”

---

## ⭐ Sistema de confianza

* Reseñas con estrellas
* Perfiles verificados
* Indicadores como:

  * “Respuesta rápida”
  * “Profesional verificado”

---

## 💎 Plan PRO (diferenciación visual)

Opciones permitidas:

* Badge “PRO”
* Borde destacado
* Mayor tamaño visual
* Prioridad en listados
* Marcadores especiales en mapa

⚠️ No debe parecer publicidad invasiva.

---

## 📱 UX/UI Principios

* Escaneo rápido de información
* Jerarquía clara (lo importante primero)
* Botones visibles y accesibles
* Feedback inmediato en acciones
* Diseño inclusivo (jóvenes + adultos + mayores)

---

## 🇦🇷 Identidad Cultural

* Lenguaje cercano (ej: “laburo”, “cerca tuyo”)
* Inclusivo para todos los oficios
* Evitar elitismo tecnológico
* Profesional pero humano

---

## 🚀 Posicionamiento

> “Chamba no es solo un marketplace.
> Es un sistema de confianza donde el trabajo tiene valor real.”

---

## 🧠 Reglas clave para IA / Diseño automático

1. Priorizar claridad sobre estética.
2. Mantener consistencia de colores.
3. No usar elementos decorativos innecesarios.
4. Cada componente debe tener propósito funcional.
5. La confianza siempre debe ser visible.
6. Evitar sobrecargar la interfaz.
7. Pensar siempre en usuarios no técnicos.

---

## 📌 Resumen

Chamba combina:

* Marketplace de servicios
* Plataforma de empleo
* Red profesional

Todo bajo un mismo concepto:

👉 **Confianza + Transparencia + Profesionalismo accesible**
